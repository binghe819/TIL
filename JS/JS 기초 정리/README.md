# 개요

자바스크립트는 이미 세계에서 제일 많이 사용하는 언어가 되었다.

우리나라에도 많은 책이 나와있지만, 각각의 책들은 서로 너무나도 다른 목차를 가지고 있다.

그래서 내가 보기 편하게 정리하기로 결심했다..
